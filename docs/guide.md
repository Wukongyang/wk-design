## 安装

```bash
$ npm install --save wk-design
# or
$ yarn add wk-design
```

## 🔨 使用

```ts
import { Button } from 'wk-design'
```

## 🤝 参与共建

```bash
$ git clone git@github.com:Wukongyang/wk-design.git
$ cd fe-funs
$ npm i
$ npm start
```

## 开发

### 1、目录结构描述

```text
src
├── index.ts // 组件库源码的入口文件
├── packageA
    └── ├── styles        // 组件 A 样式表
        ├── index.ts      // 组件 A 入口文件
        ├── index.md      // 组件 A 使用文档
        ├── PackageA.tsx  // 组件 A 文件
```

### 2、开发完成

开发分支推送到远程，创建 PR 到 master 分支
